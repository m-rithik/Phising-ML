# Synthetic Vernacular Phishing Dataset

This folder contains synthetic, human-readable phishing and benign messages in Indian vernacular languages and code-mixed text. It is meant for development and testing only.

Files:
- `sms.jsonl`: synthetic SMS-like messages
- `email.jsonl`: synthetic email-like messages (with a simple Subject line)
- `chat.jsonl`: synthetic chat messages
- `manifest.json`: counts by source

Each line is a JSON object with fields from `docs/data-schema.md`.
