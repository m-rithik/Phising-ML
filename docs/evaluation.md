# Evaluation Protocol

## Core Metrics
- Precision, Recall, F1 (overall and per-language)
- Macro-F1 across languages
- False positive rate on benign financial SMS
- Calibration error (reliability curves)

## Slices
- Language and script
- Code-mixed vs single-language
- Channel (sms/email/chat)
- Region

## Failure Analysis
- High-confidence false negatives
- High-confidence false positives
- Phishing intent confusion
